# Stone Tech OS ‚Äî Quick Start (15 min)

1. **Sign in** at `/auth` with the admin account provisioned during setup.
2. **Company profile** ‚Äî `/settings/company` (name, GSTIN, address, logo).
3. **Add first user** ‚Äî `/settings/users` ‚Üí invite ‚Üí assign role.
4. **Create a customer** ‚Äî `/customers/new`.
5. **Create an enquiry** ‚Äî `/enquiries/new`.
6. **Build an estimate** ‚Äî `/estimates/new` ‚Üí add items ‚Üí **Approve**.
7. **Send a quote** ‚Äî `/quotes/new` ‚Üí **Send PDF**.
8. **Convert to sales order** on customer approval.
9. **Raise RFQ** if procurement needed ‚Üí invite vendors ‚Üí issue PO.
10. **Send to manufacturing** ‚Üí track stages ‚Üí QC.
11. **Auto-installation** created ‚Üí team assigned ‚Üí sign-off.
12. **Invoice + receipt** ‚Üí collected in `/payments`.

Dashboards: `/dashboards/executive`, `/dashboards/analytics`, `/dashboards/forecast`, `/dashboards/collections`.
